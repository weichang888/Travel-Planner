# Travel Planner & Journal

一個以 React + Tailwind CSS 製作的 Notion 風格旅遊規劃應用。  
支援多旅程管理、行程拖曳排序、旅遊心得編輯、購物清單統計，以及本機資料持久化。

## 功能特色

- 多旅程管理
  - 新增 / 刪除多個旅程
  - 每個旅程可設定封面圖片
- 旅遊規劃（Itinerary）
  - 依天數管理行程（可新增天數）
  - 活動卡片可拖曳到任意位置調整順序
  - 活動可附圖片
- 旅遊心得（Journal）
  - 心得列表頁與編輯頁分離
  - 已發表心得可再次編輯
  - 支援多段落、段落顏色、複數圖片
- 購物清單（Shopping）
  - 「購物清單」與「編輯購物」頁分離
  - 可自訂類別（新增、重新命名、刪除）
  - 商品可附圖片（網址/上傳/拖曳）
  - 列表卡片圖片固定顯示於右側
  - 內建圓餅圖/環形圖/長條圖統計
- 待辦清單（To-do）
  - 可新增、勾選、刪除待辦事項
- 資料保護機制（避免資料消失）
  - 自動雙重保存：`localStorage` + `IndexedDB`
  - 支援「匯出備份」與「匯入備份（JSON）」手動備份還原
  - 會記住使用者名稱與所有旅程資料

## 技術棧

- React 19
- TypeScript
- Tailwind CSS
- Vite

## 快速開始

## 1. 安裝依賴

```bash
npm install
```

## 2. 啟動開發環境

```bash
npm run dev
```

預設開啟後可在瀏覽器查看本機網址（通常為 `http://localhost:5173`）。

## 3. 建置正式版本

```bash
npm run build
```

## 4. 本機預覽正式版

```bash
npm run preview
```

## Node.js 版本建議

目前 Vite 版本建議使用：

- Node.js `20.19+` 或
- Node.js `22.12+`

## 資料儲存說明

- 本專案會自動保存資料到：
  - `localStorage`
  - `IndexedDB`
- App 啟動時會自動比較時間，優先載入較新的資料版本。
- 為了跨裝置與更高安全性，建議定期使用「匯出備份」下載 JSON 備份檔。


## 備份與還原建議流程

1. 每次有重要更新後，按一次「匯出備份」。
2. 使用「匯入備份」即可還原資料。
